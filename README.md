## Setup
1. Clone repo:
   ```bash
   git clone https://github.com/rithikargb/velorent-bike-rental.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
